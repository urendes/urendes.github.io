CSS Borders

  • Border surrounds an element like a frame.
  • Border can be set with a specific:

    1. width
    2. style
    3. color

  • Example:
    p {
        border: 3px solid coral;
    }
Width
specific size in px, pt, cm, em, etc.
value using keyword       thin
value using keyword medium
value using keyword thick

Style
none Defines no border.
hidden Defines a hidden border.
dotted Defines a dotted border.
solid Defines a solid border.
dashed Defines a dashed border.
double Defines a double border.
groove   Defines a 3D grooved border. The effect depends on the border-color value.
ridge Defines a 3D ridged border. The effect depends on the border-color value.
inset Defines a 3D inset border. The effect depends on the border-color value.
outset Defines a 3D outset border. The effect depends on the border-color value.

Color
color keywords   The color names are case-insensitive. fuchsia
hex colors Hex numbers can be 0-9 and A-F. Hex values are always prefixed with a # symbol. #32a1ce
RGB Colors are specified using a value between 0 and 255 or a percentage value. rgb(128, 80, 200)
